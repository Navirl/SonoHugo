GPU Crashed or D3D Device Removed.

Use .d3ddebug to enable the D3D debug device.
Check log for GPU state information.

GPUのドライバーとUEバージョンの相性問題。
ドライバーをアップデートしたり、UEをアップデートしたり、Vulkanに切り替えたり、FPSを下げたりすると出なくなるらしい。
