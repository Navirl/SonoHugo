[生成AIとは？ChatGPTとは？ | カリキュラム](https://curriculum.sodateru-engineer.com/courses/chatgpt-course/lesson/1-1-2/)

既存のAIは分類や特徴抽出だった
生成AIは0→1、データから新たなものを作る

できること
1. 質問回答
2. 文章作成
3. 要約
4. リスト作成
5. コード生成

- 鵜呑みにしない
    - 情報は100%正確とは限らない
        - 多数決のため
- 著作権フリーと安心しない
    - 物に依っては情報の出典を明示せず、意図しない違反に繋がる
- 社内情報を迂闊に打たない
    - 設定しないと社に抜かれる

考えられる攻撃
    テキストが絡めば全部できる
- フィッシング
    - 説得力のあるメッセージ
- ソーシャルエンジニアリング
    - なりすまし
- ディープフェイク
- オートメーション
    - ボット作成

下のテンプレをそのまま使える
```
## 役割
## 命令
## 条件
```
役割にその道のプロフェッショナルな人物を指定し、その人に命令するイメージ
